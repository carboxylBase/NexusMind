| type     | 含义            |
| -------- | ------------- |
| feat     | 新功能           |
| fix      | 修复 bug        |
| docs     | 文档修改          |
| style    | 代码格式调整（不影响逻辑） |
| refactor | 代码重构          |
| perf     | 性能优化          |
| test     | 添加或修改测试       |
| build    | 构建系统修改        |
| ci       | CI 配置修改       |
| chore    | 杂项修改          |
